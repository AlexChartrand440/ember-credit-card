# Ember Credit card

This is an Ember wrapper for the amazin [Card](https://github.com/jessepollak/card) project.

## Installation
`npm install --save-dev ember-credit-card` and `ember g ember-credit-card`

## Usage

![card](http://i.imgur.com/qG3TenO.gif)

````html
{{credit-card}}
````
